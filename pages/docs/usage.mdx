

# Usage

Using **react-native-ustyle** is very simple. Just replace all the `react-native` imports with `react-native-ustyle` and you are good to go.

```diff
- import { View, Text } from 'react-native';
+ import { View, Text } from 'react-native-ustyle';
```
Here's a simple example:

```js copy
// App.tsx
import { Text, View } from 'react-native-ustyle';

export default function App() {
  return (
    <View bg="$colors$yellow" p={20} mx={20}>
      <Text c="$colors$blue">Open up App.tsx to start working on your app!</Text>
    </View>
  );
}
```

Refer this Configuration section to know what prop maps to what style.


> In `react-native-ustyle` this configuration file can be customized as per your need. This is just an example


```ts copy
// rnu.config.ts
import { createConfig } from 'react-native-ustyle';
export const CONFIG = createConfig({
  aliases:{
    p: 'padding',
    t: 'top',
    b: 'bottom',
    l: 'left',
    r: 'right',
    h: 'height',
    w: 'width',
    bg: 'backgroundColor',
    c: 'color',
    m: 'margin',
    mx: 'marginHorizontal',
  },
  tokens:{
    colors:{
      blue: '#0000FF',
      yellow: '#FFFF00',
    }
  }
} as const);

type ConfigType = typeof CONFIG;

declare module 'react-native-ustyle' {
  interface ICustomConfig extends ConfigType {}
}
```
